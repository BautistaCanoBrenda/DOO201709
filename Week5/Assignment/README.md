# Asignación de la semana 5

## Crear un login simple con MVC

Objetivo: Al finalizar la actividad deberás ser capaz de construir un sistema de autenticacin simple basado en el patrón de diseño Modelo-Vista-Controlador.

## Preparación
La descripción de esta asignación parte del supuesto que ya cuentas con la versión de NetBeans (EE) y que incluye un contenedor Web GlassFish o Apache Tomcat atendiendo un puerto de escucha 8080.

## Actividades

En esta asignación se realizará lo siguiente:

Crear una mini-aplicación web que funcione con el patrón MVC:
- Crear tres páginas JSP para que funcionen como vistas (`Views`).
- Crear un servlet que funcionará como controladores ('Controller).
- Crear dos clases de Java que funcionarán como modelos (`Model`).
